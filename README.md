# Iris Dataset Visualization

## Objective
The objective of this project is to explore, analyze, and visualize the Iris dataset using Python data analysis and visualization libraries.

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Scikit-learn

## Visualizations
- Scatter Plot
- Histogram
- Box Plot

## Features
- Dataset inspection
- Statistical analysis
- Feature relationship visualization
- Outlier detection

## Result
Successfully explored and visualized the Iris dataset and analyzed feature relationships, distributions, and outliers using multiple visualization techniques.
